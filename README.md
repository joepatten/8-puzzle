# 8-Puzzle Game
## About
The goal of this game is to move the "blank" tile in order to get the following tileset:

+---+---+---+
| 1 | 2 | 3 |
+---+---+---+
| 4 | 5 | 6 |
+---+---+---+
| 7 | 8 |   |
+---+---+---+
